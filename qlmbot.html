02 — Conception
Engineered for
precision.
01
Entièrement asynchrone
Requêtes réseau parallèles via aiohttp. Scan de 23 ports, recherche sur 600+ sites — en quelques secondes sans bloquer le bot.
02
Données persistantes
SQLite3 intégré. Protections, whitelist, flags, tâches et configurations survivent aux redémarrages sans configuration externe.
03
Confidentialité by design
Tokens, mots de passe et résultats sensibles transmis exclusivement en message privé. Le message public est supprimé immédiatement.
04
Interface interactive
Pagination automatique, tableaux de bord temps réel, boutons de confirmation, sélecteurs de catégories — Discord natif.
05
Architecture modulaire
Chaque module de protection s'active indépendamment. Une seule commande !secur max active tout simultanément.
06
Sources multiples
ipapi.co · HackerTarget · GitHub API · Open-Meteo · Guerrillamail · Wayback Machine · binlist · macvendors — agrégés nativement.
